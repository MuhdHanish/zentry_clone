## Installation

Clone the repository and install dependencies:

```sh
git clone https://github.com/MuhdHanish/zentry_clone.git
cd zentry-clone
npm install
```

## Running the Project

Start the development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```
